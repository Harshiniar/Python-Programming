Questions Taught in class:
Write a program to print Hello World with tab
Write a program to get a character as an input and print its ASCII value.
Financial Application for Rounding Values
Write a program that takes two numbers as input from the user and performs basic mathematical operations on them, such as:Square root,power,absolute of two numbers.
Write a Pyhton program to print Students Information such as Name,age,cgpa and grade.
Given the number of newspapers sold a, the selling price per copy b, and the printing cost per copy c, write a program to calculate the profit earned by The Herald, considering a fixed overhead cost of Rs.100
Given an integer N representing workout duration in minutes, write a program to return the equivalent duration in seconds.
Given the total number of friends and the number of teams, write a program to determine how many friends will be in each team and how many friends will be left out after forming equal teams.
Given an integer n (1–9) representing a user-selected position in a 3×3 Tic-Tac-Toe board, write a program to determine the corresponding row and column indices (0-based) in a 2-dimensional array.
Given a four-digit integer, write a program to find and print the sum of its first and last digits, ignoring any sign.
Given the coordinates of Mani’s house(x1,y1) and Kumar’s house (x2,y2), write a program to find and display the midpoint representing Arun’s house, formatted to one decimal place.

Questions for Slow Learners:
Write a program to print the Hello World with a new line
Write a program to get a number (ASCII value) as input and print its equivalent character.
Write a program to read a user’s name and membership type and print them on separate lines.
Given two monetary amounts represented by dollars and cents (d1,c1) and(d2,c2), write a program to calculate and display the total amount in dollars and cents, handling any overflow from cents to dollars.
Given the total number of gold coins, Long Ben’s share percentage, and Blackbeard’s share percentage, write a program to calculate the number of coins received by Long Ben, Blackbeard, and each of the remaining three pirates, discarding any leftover coins.
Given the total step target S, the steps already completed D, and the number of days N, write a program to calculate and return the rounded average number of steps per day the user needs to take to reach the target.
Given the principal amount X, rate of interest R%, and loan period Y years, write a program to calculate the total interest, total amount payable, 2% discount on the interest, and the final settlement amount, all formatted to two decimal places.
